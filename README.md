# Multi-Functional Python Toolkit

## Project Overview

The **Multi-Functional Python Toolkit** is a menu-driven Python application that combines several useful utilities into a single program. It demonstrates the use of Python built-in modules, custom packages, file handling, and modular programming concepts.

The application allows users to perform date and time operations, mathematical calculations, random data generation, UUID creation, module exploration, and business operation simulations while maintaining logs of all activities.

---

## Features

### 1. Date & Time Operations
- Display the current date and time in a custom format.
- Calculate the difference between two dates.
- Simple stopwatch implementation.

### 2. Math Utilities
- Calculate sine values.
- Find logarithms.
- Calculate factorials.
- Compute compound interest.
- Calculate the area of a circle.

### 3. Randomization & Simulation
- Generate secure random passwords.
- Generate 6-digit OTPs.
- Randomly sample data from a dataset.
- Play a simple Higher-or-Lower guessing game.

### 4. UUID Generation
- Generate secure UUID version 4 identifiers.
- Create sample invoice IDs using UUIDs.

### 5. Dynamic Module Exploration
- Explore Python modules using the `dir()` function.
- Inspect built-in modules like:
  - math
  - random
- Inspect custom modules.

### 6. Business Use Case Simulation
- Simulate employee shift tracking.
- Generate employee passwords.
- Create invoice IDs.
- Save transaction logs.
- Display available toolkit features.

### 7. Log Management
- Save all application activities to a text file.
- Read previously saved logs.
